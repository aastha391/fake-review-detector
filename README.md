# Fake Review Detector

An ML-powered web app to detect fake product reviews using Linear SVM.

## 🚀 Live Demo
- **Frontend:** https://fake-review-detector-pink.vercel.app
- **Backend API:** https://fake-review-detector-v4vz.onrender.com
- **API Docs:** https://fake-review-detector-v4vz.onrender.com/docs

## 🛠️ Tech Stack
- Python, scikit-learn, FastAPI, React, Vite, TF-IDF, Linear SVM

## 📊 Model Performance
- Accuracy: 91.12%
- ROC-AUC: 0.9727
- F1-Score: 0.9112
